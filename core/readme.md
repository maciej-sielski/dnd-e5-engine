# @ms-sol/dnd-e5-engine

Basic interfaces and functions to help you build your own RPG games. Based on Dungeons & Dragons 5e.

## Installation

Install the library using npm or yarn:

```bash
npm install @ms-sol/dnd-e5-engine
```
